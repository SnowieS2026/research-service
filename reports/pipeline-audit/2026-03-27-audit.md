## Pipeline Architecture Notes

- Entry point: `npm run build && npm run pipeline` (or `npm run agents` for agent mode)
- Agents communicate via JSONL queues at `logs/agent-queue/*.queue.jsonl`
- AdaptationAgent scores strategies per program in `logs/agent-state/adaptation-store.json`
- ScannerAgent handles nuclei/dalfox/sqlmap execution
- BrowserAgent handles Playwright session management
- ReporterAgent compiles findings into reports

**Key files:**
- `src/agents/ScannerAgent.ts` — nuclei/dalfox/sqlmap wrapper
- `src/agents/BrowserAgent.ts` — Playwright session management
- `src/agents/CoordinatorAgent.ts` — orchestrates other agents
- `src/agents/DiscoveryAgent.ts` — program discovery
- `src/scripts/daily-discovery.js` — standalone discovery (cron-disabled but functional)
